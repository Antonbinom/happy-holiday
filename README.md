# Сервис поздравлений 

## Описание

Pet-проект React SPA.

## Функционал 

- Загрузка данных с сервера
- Динамическая отрисовка данных в случайном порядке

## Стек технологий

- React
- Redux - хранилище состояний компонентов
- React-router - загрузка страниц без перезагрузки
- React-hooks
- Custom-hooks
- Axios - Rest-API
